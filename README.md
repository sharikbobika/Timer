Timer
Timer with settings v1. This is the first version of my timer application!

Getting Started
Follow these steps to set up and run the project locally on your machine.

1. Installation
   First, get the source code using one of the following methods:

 1) Via Git: Clone the repository:

git clone https://github.com/sharikbobika/Timer.git

2) Via ZIP: Download the repository as a ZIP archive, extract it to a folder of your choice, and open that folder.

2. Installing Dependencies
Open your terminal (Command Prompt or PowerShell) inside the project folder. To do this quickly, you can open the folder in File Explorer, click on the path bar at the top, type cmd, and press Enter.

Run the following command to install all required libraries:

pip install -r requirements.txt

3. Running the Application
Once the installation is complete, start the application by running:

python app.py

After the server starts, you can access it in your web browser at:
👉 http://127.0.0.1:5000/

Prerequisites
Ensure you have Python installed on your system. You can verify this by running python --version in your terminal.

Начало работы
Следуйте этим шагам, чтобы настроить и запустить проект локально на вашем компьютере.

1. Установка
Сначала получите исходный код одним из следующих способов:

1) Через Git: Клонируйте репозиторий:

git clone https://github.com/sharikbobika/Timer.git

2) Через ZIP: Скачайте репозиторий в виде ZIP-архива, распакуйте его в любую удобную папку и откройте её.

2. Установка зависимостей
Откройте терминал (командную строку или PowerShell) внутри папки с проектом. Чтобы сделать это быстро, откройте папку в Проводнике, кликните по строке пути вверху, введите cmd и нажмите Enter.

Выполните следующую команду для установки всех необходимых библиотек:

pip install -r requirements.txt

3. Запуск приложения
После завершения установки запустите приложение, выполнив команду:

python app.py

Как только сервер запустится, вы сможете открыть его в браузере по адресу:
👉 http://127.0.0.1:5000/

Требования
Убедитесь, что на вашем компьютере установлен Python. Проверить это можно, выполнив команду python --version в терминале.
